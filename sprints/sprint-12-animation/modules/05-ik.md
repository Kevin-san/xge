# 反向运动学模块 (IK)

## 模块概述

反向运动学（IK）模块负责实现骨骼链的逆向求解，支持两骨 IK、CCD（循环坐标下降）、FABRIK 等经典算法，并提供瞄准 IK、注视 IK 和脚部 IK 等高级功能。本模块用于实现角色与环境的自然交互。

## 优先级

**P0（核心）：**
- IK::two_bone_ik 两骨 IK
- IKChain 骨骼链管理
- AimIK 瞄准 IK

**P1（重要）：**
- IK::ccd_ik CCD 迭代求解
- LookAtIK 注视 IK
- FootIK 脚部贴合

**P2（增强）：**
- IK::fabrik FABRIK 算法
- 多目标 IK
- IK 约束系统
